root.aspect(minNumer, minDenom, maxNumer, maxDenom)
If minNumer, minDenom, maxNumer, and maxDenom are all specified, then they will be passed to the window manager and the window manager should use them to enforce a range of acceptable aspect ratios for window. The aspect ratio of window (width/length) will be constrained to lie between minNumer/minDenom and maxNumer/maxDenom. If minNumer etc. are all specified as empty strings, then any existing aspect ratio restrictions are removed. If minNumer etc. are specified, then the command returns an empty string. Otherwise, it returns a Tcl list containing four elements, which are the current values of minNumer, minDenom, maxNumer, and maxDenom (if no aspect restrictions are in effect, then an empty string is returned). Same as wm_aspect
root.attributes(option, value, option, value, ...)
-alpha float
Controls opacity from 0.0 to 1.0. May not work on any Linux devices.

-disabled bool
Makes it impossible to interact with the window and redraws the focus.

-fullscreen bool
Controls whether the window fills the whole screen, including taskbar.

-modified bool
Controls whether the window is marked as being modified (different close button).

-notify bool
Controls whether the window's icon bounces in the dock.

-titlepath filename
Associates a (draggable) file icon with the window title that can be used to manage the given file with other applications.

-toolwindow bool
Makes a window with a single close-button (smaller than usual) on the right of the title bar.

-topmost bool
Makes sure the window always stays on top of all other windows.

-transparent bool
Controls whether the window has a shadow (useful for transparent windows, though care must be taken to draw appropriately to take advantage).

-type list
This is provided in Tk 8.6, 8.5.9+ and 8.4.20+ to support the extended window manager hint property that governs a toplevel window's type on X11. Options:
  • combo
  • dnd
  • desktop
  • dialog
  • dock
  • dropdown_menu
  • menu
  • normal
  • notification
  • popup_menu
  • splash
  • tooltip
  • toolbar
  • utility


-zoomed bool
Controls whether this window is in the zoomed state (because root.state('zoomed') doesn't work on X11).

Same as wm_attributes
