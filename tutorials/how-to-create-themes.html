Creating a ttk theme

You'll need:

  • Inkscape, or some other usable graphic design software
  • Some kind of familiarity with the Tcl programming language

Getting started

In this tutorial we'll create a ttk theme and by the end you will learn a lot of tips and tricks about them. We'll make this theme in Tcl, because although it's possible in Python with the ttk.Style object, it has a couple of bugs and an awful idiot syntax, so I don't recommend it.

Ttk themes are not dynamically rendered, so we canâ€™t just tell to Tk what we want and then it does it for us. Instead, we create images in Inkscape (or in another program) and export them as png images, then import them into Tk and apply them to the widgets.

Let's get into it

First, start by calling theme create, which will actually create the theme. 

ttk::style theme create some_nice_theme_name -parent clam

Usually we use a process like this to load the images into the images variable, which is a dictionary. Later we can access this images from this dict. 

proc load_images {imgdir} {
    variable images
    foreach file [glob -directory $imgdir *.png] {
        set images([file tail [file rootname $file]]) \
        [image create photo -file $file -format png]
    }
}

# And then run the process, to load them
load_images [file join [file dirname [info script]] your_image_dir]

The script will go through the imgdir library, and create a photo image from each of them.
